# Wanderlust 🏡

Wanderlust is a full-stack web application inspired by Airbnb that allows users to explore, list, and book unique accommodations. The platform provides a seamless experience for travelers looking for places to stay and property owners who want to rent out their villas and homes.

## 🚀 Features

### User Authentication

* Secure user registration and login
* Session-based authentication
* Authorization for protected routes

### Property Listings

* Browse available villas and accommodations
* View detailed property information
* Add new property listings
* Edit and delete existing listings

### Booking Experience

* Explore accommodations with pricing details
* View property descriptions and images
* Easy-to-use booking interface

### Reviews & Ratings

* Leave reviews for listed properties
* Read feedback from other users
* Improve trust and transparency

### Responsive Design

* Mobile-friendly interface
* Clean and intuitive user experience

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/wanderlust.git
cd wanderlust
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory and add:

```env
ATLASDB_URL=your_mongodb_connection_string
SECRET=your_session_secret
```

### Run the Application

```bash
node app.js
```

or

```bash
npm start
```

The application will run on:

```bash
http://localhost:3000
```
